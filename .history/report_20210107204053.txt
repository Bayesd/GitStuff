•	Beskriv hur du använde dig av Git i inlämningsuppgiften. Vilka Git-kommandon använde du och varför?
    - mkdir för att skapa mappen på datorn
    - cd till mappen
    - git init för att initera mappen, dvs göra den till ett repo
    - Skapade mina filer i VSCode
    - git add --all för att lägga till filerna inför committ
    - git commit -m "..." för att commita, -m för att lägga till meddelande
    - git push för att pusha till repot.
    - git log för att se förändringar som gjorts
    - git branch ... för att skapa en ny branch
    - git checkout ... för att byta branch
    - git push --set-upstream origin {ny branch} för att göra en merge med main
